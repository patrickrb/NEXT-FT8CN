About Auto-Follow CQ
  Auto-Follow CQ: When FT8CN decodes a CQ message, it will push the message to the "Call" screen's message list, making it easy to respond to CQ messages from the Call screen. Auto-followed CQ callsigns are not permanently saved to the followed callsign database.
About Auto-Call Followed Callsigns
  When a target callsign is set to "followed," FT8CN will automatically call that callsign when it sends a CQ.
  FT8CN permanently saves your followed callsigns. Each time you use FT8CN, new messages from previously followed callsigns will be pushed to the "Call" screen's list. To unfollow a callsign, you can delete it through the web interface (access FT8CN via a browser on the local network).
