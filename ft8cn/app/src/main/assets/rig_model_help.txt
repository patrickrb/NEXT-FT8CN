About Radio Model: Based on the selected radio model, FT8CN uses the appropriate command set to control the radio. Different brands of radios use different command sets, so please select the radio you are currently using from the list.
For ICOM series radios, this list includes models with known default CI-V addresses and maximum supported baud rates. By selecting a radio from this list, the CI-V address and baud rate will be set automatically. If your radio model is not in this list, you will need to manually set the appropriate CI-V address and baud rate.
